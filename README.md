# demo-collection

## Project Introduction 项目介绍

#### 这个项目是自己闲暇的时间做的，里面包含了一个m3u8播放器（推流播放视频），还有几个360度全景的东西，和一些VR的页面，自己就是非常简单的实现了一个VR看房的效果（手机可以看VR）,还有一些cesium相关的简单操作，后续应该会增加几个大数据大屏，其他的还不知道会有什么
